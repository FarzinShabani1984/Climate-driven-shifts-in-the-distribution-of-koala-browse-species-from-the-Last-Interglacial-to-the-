This folder contains data for the Australian topsoil clay fraction (% weight) sourced from the Harmonized World Soil Database ver. 1.2 (Nachtergaele et al. 2009) at a resolution of 2.5 minutes.

We built our model using these data, but we are not redistributing the Harmonized World Soil Database dataset. 

Refer to Nachtergaele, F. et al. 2009. Harmonized world soil database. – ISRIC, Wageningen
